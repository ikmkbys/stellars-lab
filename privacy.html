プライバシーポリシー

最終更新日：2026年3月27日

Stellars Lab（以下「当スタジオ」）は、提供するプロダクト（Notipop・Polaris）における個人情報の取り扱いについて、以下のとおりプライバシーポリシーを定めます。

第1条（取得する情報）

当スタジオは、各プロダクトにおいて以下の情報を取得することがあります。

【Notipop（Chrome拡張）】

  • ユーザーが手動で選択したテキスト・ページタイトル・URL（メモとして送信する際のみ）
  • Notion API アクセストークン（ユーザーが設定画面で入力したもの。ブラウザのローカルストレージに保存）
  • 保存先として指定された Notion データベース ID

【Polaris（Webアプリ）】

  • Google アカウント情報（Google Sign-In 経由。表示名・メールアドレス・UID）
  • ユーザーが入力した目標・中期目標・アクションテキスト
  • アプリ設定（テーマ・言語・AIトーン）
  • 決済情報（Stripe を介して処理。当スタジオはカード情報を保持しません）

第2条（利用目的）

取得した情報は以下の目的にのみ使用します。

  • 各プロダクトの機能提供（Notion へのメモ保存、目標管理データの表示など）
  • AIによる目標ロードマップ・月次レポートの生成（Polaris のみ）
  • 決済処理および有料プランの管理（Polaris のみ）
  • サービス改善・障害対応

第3条（第三者提供）

当スタジオは、以下の場合を除き、ユーザーの個人情報を第三者に提供しません。

  • ユーザーの事前同意がある場合
  • 法令に基づく場合

なお、各プロダクトは以下の外部サービスを利用しており、それぞれのプライバシーポリシーが適用されます。

  • Notion API（Notipop・Polaris）— Notion プライバシーポリシー
  • Firebase / Google Cloud（Polaris）— Google プライバシーポリシー
  • Anthropic Claude API（Polaris）— Anthropic プライバシーポリシー（ユーザーデータは AI 学習に使用されません）
  • Stripe（Polaris）— Stripe プライバシーポリシー

第4条（データの保存場所）

Notipop：Notion API アクセストークン・データベース ID はユーザーのブラウザ内ローカルストレージにのみ保存され、当スタジオのサーバーには送信・保存されません。

Polaris：目標・アクションデータは Google Firestore（データセンター：米国）に保存されます。

第5条（Cookieおよびトラッキング）

現時点において、当スタジオは広告目的の Cookie やトラッキングツールを使用していません。Polaris では Firebase Authentication のセッション管理のために Cookie を使用することがあります。

第6条（データの削除）

Notipop：拡張機能をアンインストールすると、ローカルストレージに保存されたデータは削除されます。

Polaris：アプリ内「設定 → アカウント削除」から、Firestore に保存されたすべてのデータおよびアカウントを削除できます。

第7条（未成年者の利用）

当プロダクトは13歳未満の方の利用を想定していません。13歳未満の方の個人情報を意図せず取得した場合、速やかに削除します。

第8条（プライバシーポリシーの変更）

本ポリシーは必要に応じて変更されることがあります。変更後のポリシーは本ページへの掲載をもって効力を生じます。重要な変更がある場合は各プロダクト内でお知らせします。
